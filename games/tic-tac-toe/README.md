# Tic-Tac-Toe

Tic-Tac-Toe is a simple strategy game played on a 3Ã—3 grid. Players take turns marking spaces with X or O, trying to get three in a row.

## Game Overview

- **Players**: 2
- **Board Size**: 3Ã—3 grid
- **Pieces**: X and O marks
- **Objective**: Get three marks in a row (horizontal, vertical, or diagonal)

## Rules Summary

Players alternate placing their mark (X or O) in empty squares. The first player to get three of their marks in a row wins. If all squares are filled without a winner, the game is a draw.
